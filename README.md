        Projeto Prático em SQL Server – Modelagem e Análise de Vendas

Recentemente desenvolvi um banco de dados relacional completo utilizando SQL Server, aplicando conceitos fundamentais de modelagem e manipulação de dados.

 Criação e configuração do banco de dados (controle de crescimento MDF/LDF)
 
Modelagem relacional com definição de Primary Keys e Foreign Keys

 Implementação de integridade referencial

 Inserção, atualização e exclusão de dados (CRUD)
 
 Controle de Identity com DBCC CHECKIDENT

 Consultas com JOINs para cruzamento de dados

 Subqueries no SELECT e FROM

Agregações com GROUP BY para análise de vendas por período



          O projeto simula um ambiente de vendas contendo:

   Clientes
   
   Produtos
   
   Regiões
   
   Vendas

          Com isso, foi possível gerar indicadores como:

 Total de vendas por ano
 
 Quantidade vendida
 
 Análise por categoria de produto
 
 Consolidação de dados via JOIN
